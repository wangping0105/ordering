== README
公司加班的订餐系统.
因为是公司内部使用, 所以也不需要密码了,输入用户名即可.

`注意用户表是 order_user`

* Ruby 2.0.0

* System dependencies

* Configuration

* Database
    db:create
    db:migrate
    db:seed
    rails s
    link to http://localhost:3000


Please feel free to use a different markup language if you do not plan to run
<tt>rake doc:app</tt>.
